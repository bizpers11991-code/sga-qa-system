# Word Template Design Guide for Nice PDFs
## Template Structure:
- Header: SGA Logo, Report Title, Date
- Sections: Job Details, Daily Report (with tables), Asphalt Placement (repeatable rows), Photos
- Footer: Page numbers, SGA branding
- Styling: Orange accents (#FF6600), clean fonts, professional layout

## PDF Generation Checks:
- Ensure Word template has content controls for all fields.
- Test flow: Populate template → Convert to PDF → Save to SharePoint.
- Issues: If PDF fails, check file permissions; add error handling in flow.
